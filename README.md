# Interactive Lung Mechanics Activity

This folder contains a GitHub Pages–ready interactive learning activity for teaching respiratory mechanics.

## What's included
- `index.html` — the main self-contained interactive webpage
- `assets/` — infographic-style lung images used in each interactive panel

## Topics covered
1. Boyle’s Law
2. Pascal’s Law
3. Poiseuille’s Law
4. Continuity Equation
5. Parallel Airways
6. Bernoulli Effect
7. Equal Pressure Point (EPP)
8. Positive Expiratory Pressure (PEP)

## How to use
### Option 1: Upload to GitHub Pages
1. Create a new GitHub repository.
2. Upload **all files and the `assets` folder**.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the main branch and `/ (root)` folder.
6. Save — GitHub will publish the site.

### Option 2: Run locally
Just open `index.html` in a browser.

## Teaching idea
Ask students to move one slider at a time and explain whether the main effect is on:
- pressure generation
- pressure transmission
- resistance
- airflow velocity
- airway collapse risk
